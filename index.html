Web Component based pattern library

Version 1

This is the initial version of the pattern library. The structure of the patterns is defined in custom Polymer elements while the styles are defined in a master CSS file. I hope to get the styles moved inside the patterns themselves but for now I want to get the patterns working. For a production application we will either use the patterns as-is and compose our templates and pages.
